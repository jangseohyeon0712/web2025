JavaScript 합계 예제

JavaScript: 1부터 10까지 합 구하기


function calcSum() {
  let sum = 0;
  for(let i = 1; i <= 10; i++) {
    sum += i;
  }
  console.log(`1부터 10까지 더하면 ${sum}입니다.`);
}

calcSum();
▶ 1부터 10까지 더하면 55입니다.
