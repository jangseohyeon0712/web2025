Chapter 8 summary 최상위 객체, window

내장 객체란

- 사용자가 가져다 쓸 수 있도록 미리 만들어진 객체.

- 웹 브라우저에 웹 문서가 열리면 가장 먼저 window 객체가 만들어지고 하위에 웹 브라우저 요소에 해당하는 객체들이 만들어진다.

- window 객체를 비롯해 하위에 연결된 객체들은 모두 HTML 웹 API에 만들어진 객체들

window
├── document
│   ├── anchor
│   ├── form
│   ├── image
│   ├── button
│   └── text
├── history
├── location
├── navigator
├── screen
├── array
├── date
├── function
└── math
